# Overview

This repository is a personal portfolio site (Next.js, React, Tailwind). The hero includes a deferred WebGL “brain” wireframe with separate **mesh scale** and **neural orb** (point-sprite) rendering.

**Internal specification:** The five files in `docs/` are for maintainers and tooling. They are **not** exposed as public site routes and must not be linked from production UI (see `docs/REQUIREMENTS.md`).
